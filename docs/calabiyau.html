Calabi Yau Manifold

A Calabi Yau Manifold is a Complex surface on which we can assign a Ricci-flat (Kahler) metric. This has been shown to be equivalent to it having \(SU(n)\) holonomy (this describes the way arrows are transformed when moved in a loop). All smooth algebraic varieties in \( \mathbb{CP}^n\) are Kahler manifolds.
